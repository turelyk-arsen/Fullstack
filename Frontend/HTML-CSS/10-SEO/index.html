Best kept secret to rank on Google

Process

  1. Code a cool website
  2. Host the website online: Netlify is the cheapest (sign up using GitHub) and link it to a domain name (DNS)
  3. Register to Google https://search.google.com/
  4. Search engine optimization: https://developers.google.com/search/docs/fundamentals/seo-starter-guide

SEO

  1. sitemap.xml
  2. robots.txt
  3. Logic Heading (h1, h2, ..)
  4. Good actual content (duh...)
  5. Some meta for sharing (OpenGraph...)
